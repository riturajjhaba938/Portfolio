Check out the live version of my portfolio here!

## ✨ Features

- **Modern UI/UX:** Designed with a sleek, dark-themed, and responsive interface.
- **Smooth Animations:** Built using `framer-motion` for fluid page transitions and scroll animations.
- **Interactive Background:** Features a custom particle background for an engaging visual experience.
- **Custom Cursor:** Enhanced interactivity with a tailored custom cursor.
- **Fully Responsive:** Adapts seamlessly to all screen sizes (mobile, tablet, desktop).

## 🛠️ Built With

- **[React 19](https://react.dev/):** Library for web and native user interfaces.
- **[Vite](https://vitejs.dev/):** Next Generation Frontend Tooling for fast builds.
- **[Framer Motion](https://www.framer.com/motion/):** Production-ready animation library for React.
- **[Sass](https://sass-lang.com/):** CSS with superpowers for advanced and maintainable styling.
- **[React Icons](https://react-icons.github.io/react-icons/):** Comprehensive icon library.

## 📂 Project Structure

- `src/components/`: Contains all modular React components (`Hero`, `VisionaryAbout`, `Gallery`, `EducationSkills`, `Contact`, etc.)
- `src/hooks/`: Custom React hooks (e.g., `useSmoothScroll`).
- `src/App.jsx`: Main application layout assembling all sections.
- `index.html`: Entry point of the application.

## ⚙️ Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/riturajjhaba938/Portfolio.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd Portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Build for production:**
   ```bash
   npm run build
   ```
